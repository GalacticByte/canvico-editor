# Function: validateFile()

> **validateFile**(`file`, `maxFileSizeMB`): `void`

Validates if the selected file is a supported image and within the size limit.

## Parameters

### file

`File`

The file to validate.

### maxFileSizeMB

`number`

The maximum allowed file size in megabytes.

## Returns

`void`

## Throws

If the file type is unsupported or the file size exceeds the limit.
