Class: DOMManager

Manages all DOM interactions for the CanvicoEditor. It queries and validates the existence of all required and optional elements upon instantiation, providing a single, reliable source of DOM nodes.

Constructor​

Creates an instance of DOMManager.

Parameters​

The main editor configuration options containing CSS selectors.

Returns​

Throws​
